HelloBookWorms

This site is the result of a MEAN stack experiment. The main theme of this site is to share books and collaborate with your thoughts on any book item. The text stack used is

  • Mongodb :For saving document data
  • Express :For routing the API calls
  • Angular :Framework to handle front end data
  • NodeJS :Processing the HTTP requests acting as an API
  • Socket.IO :Real-time interactions in Chats page. If User A and User B are in same chat page and posting a comment it will be loaded without page reload

This site has been planned to have 3 Phases of development as described bellow.

Phase 1

This phase majorly focuses on lending and sharing books with all users.

  • Access: Registration and login to site
  • Contribute: Lend your book and list it to other users for viewing
  • Collaborate: Public forums. Place where people can dive in to discuss about any topic referring to a book

Phase 2

This phase majorly focuses on making users get connected strongly by forming network.

  • Networks: Add users to your network and manage your contents across.
  • Private forums: Where people can create private discussions that can be viewed only by those users in the network

Phase 3

This phase majorly focuses on users being well informed about the topics that are falling under their interests radar.

  • Notifications: on book request to borrow from you.
  • Alerts:.Adding favourite topics/genres/interests as tags that will let us perform search on the related events happening in the site
  • Messages:Moving away from traditional emailing. Send private messages manages per profile in order to have ease in communication

Phase 4

This phase majorly focuses on mobilising the application with a Cordova based mobile App(If only the site hits more user base).
